### Pré Requisitos

- `$ python3 -m venv .venv`
- `$ source .venv/bin/activate`
- `$ pip install -r requirements.txt`

### Instalação

- `$ ./manage.py migrate`
- `$ ./manage.py createsuperuser`
- `$ ./manage.py runserver`

### Comandos adicionais

- Ao instalar novos pacotes: `$ pip freeze > requirements.txt`

### Tecnologias

- Django Rest Framework
- SQLite
